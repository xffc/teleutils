## TeleUtils
A simple Telegram bot with the tools I use

## Requirements
1. A pair of hands (preferably normal ones)
2. [JRE 25](https://docs.aws.amazon.com/corretto/latest/corretto-25-ug/downloads-list.html)
3. [Bot token](https://core.telegram.org/bots/features#botfather)
4. [Distribution of teleutils](#building)

## Building
1. Clone project `git clone https://github.com/xffc/teleutils`
2. Run `gradlew build`
3. Grab zip/tar from build/distributions

## Launching
In the `bin` folder (from the distribution you built), run `teleutils TOKEN`

`TOKEN` is the token you received from the BotFather.
